# Cocuk Takip Sistemi

Çocuk Takip Programı, Siz evde yokken çocuğunuzun bilgisayar başında yaptıklarını görüntülemenizi ve kendinize mail gönderilmesini sağlayabileceğiniz bir bilgisayar programıdır...
